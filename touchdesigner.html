TouchDesigner Experiments

Visual Jockey & Live Stream Art

About This Project

These TouchDesigner experiments represent a series of visual explorations captured during live streams and performances. Using real-time visual programming, these pieces combine generative art, reactive visuals, and interactive elements to create immersive digital experiences. Each experiment pushes the boundaries of what's possible with procedural generation and real-time rendering.

The work spans from abstract audiovisual compositions to data-driven visualizations and interactive installations. TouchDesigner's node-based workflow allows for rapid prototyping and live manipulation of visual parameters, making it ideal for performance art and VJ sets. These experiments showcase the integration of shader programming, particle systems, and real-time effects processing to create dynamic, ever-evolving visual landscapes.
