Performative Cyber-Physical Wearable

About This Project

ATLTVHEAD is a Performative Cyber-Physical wearable suit that transforms the city into a hybrid digital-physical playground. Using custom-designed wearable robotics and IoT architecture, this project creates a two-way interface between the internet and the real world. The suit adapts cutting edge technology to be human-focused, using engineering not just to solve problems, but to spark community, curiosity, and spontaneous joy in public spaces.

The project combines robotics, wearable technology, IoT systems, and interactive art to create immersive experiences that bridge the gap between academic research and public art. Each performance is a unique exploration of how technology can enhance human connection and create playful, engaging moments in urban environments.
